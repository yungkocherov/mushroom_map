# data/copernicus/

Сюда кладутся скачанные GeoTIFF'ы Copernicus HRL.

Как скачивать — см. [../../docs/copernicus_download.md](../../docs/copernicus_download.md).

```
tree_species/    ← классификация по породам (DLT или Tree Species)
tcd/             ← опционально: Tree Cover Density
class_map.yaml   ← опционально: кастомный маппинг кодов классов на slug'и
```

Файлы в этой папке в git не коммитятся (см. .gitignore).
