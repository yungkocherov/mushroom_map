# Журнал изменений

История значимых изменений Geobiom — для тех, кто заходит после паузы и хочет понять, что нового. Технические коммиты — в [GitHub-репозитории](https://github.com/yungkocherov/mushroom_map/commits/main); здесь — только то, что меняет опыт пользователя.

## 2026-04 · Редизайн «Geobiom»

- **Бренд.** Проект переименован: `mushroom-map` → **Geobiom**. Tagline — «лес ленобласти». Домен `geobiom.ru` подключен.
- **Главная страница теперь карта.** Вместо hero-секций с виджетами — choropleth по 18 районам Ленобласти, sidebar слева с прогнозом грибности и топом видов на выбранный день.
- **Прогноз грибности (preview).** API `/api/forecast/at` и `/api/forecast/districts` отдаёт детерминированный fixture; ML-модель из сестринского репо `mushroom-forecast` приедет позже и заменит источник без изменений во фронте.
- **Spotlight (⌘K).** Глобальный поиск по видам и топонимам Ленобласти.
- **Реструктуризация навигации.** Четыре раздела: «Карта», «Виды», «Споты», «Методология». Старые URL (`/map`, `/forecast`, `/about`, `/cabinet/spots`, `/guide`) — 301'ы на новые места.
- **`/species/:slug` редизайн.** Hero с фото-фоном и градиентом, двухколоночное тело с сезоном, сродством к лесу и CTA «Открыть на карте».
- **`/methodology` хаб.** Четыре рубрики: источники данных, модель прогноза, о проекте, юридическое.

## 2026-04 · Лесные и геофичи

- **Полигоны Рослесхоза — ~2 млн выделов.** Полное покрытие Ленобласти из ФГИС ЛК.
- **Почвенная карта.** Полигоны Докучаевского ин-та (1:2.5М) — слой и `/api/soil/at` для будущей feature-extractor'ной модели.
- **Водотоки OpenStreetMap.** ~204k линий с расстоянием до воды через KNN.
- **Рельеф Copernicus GLO-30.** DEM/slope/aspect; цветной hillshade-слой с гипсометрией.
- **Районы ЛО.** 18 muni-районов из OSM Overpass, choropleth-слой `/api/districts/`.
- **Gazetteer + Natasha NER.** ~21k топонимов; VK-посты привязываются к району текстовым анализом.

## 2026-03 · VK-пайплайн

- **69k постов из ВК-сообществ грибников.** Корпус с 2014 года, четыре стадии обработки.
- **Qwen3.5-9B классификатор фото.** Заменил Gemma 3 12B; 18 видов + 3 ягоды; стабильные prompt-версии в `pipelines/prompts/`.
- **Агрегаты «что сейчас собирают».** На главной и в районах — топ видов с датой и долей.
